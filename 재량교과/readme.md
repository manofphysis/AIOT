# AIoT 반도체 설계 과정 재량교과 과목 내용

AIoT(Artificial Intelligence of Things) 반도체 설계 과정에서는 융합형 인재 양성을 위해 다양한 재량교과 과목이 개설될 수 있습니다. 이 과목들은 실습 중심의 수업으로 구성되며, 학생들이 직접 회로를 설계하고, IoT 기기를 제작하고, AI를 적용하는 경험을 쌓을 수 있도록 설계됩니다.

---

## 📘 재량교과 과목 예시

| 과목명 | 주요 내용 | 목표 |
|--------|-----------|------|
| **AI와 IoT 기초** | 센서 데이터 수집, 통신 모듈 활용 (WiFi, BLE), Arduino 프로그래밍 | AIoT의 기본 구조와 원리를 이해하고 간단한 장치를 제어할 수 있다. |
| **임베디드 시스템 설계** | 마이크로컨트롤러(MCU) 이해, GPIO 제어, 실시간 데이터 처리 | IoT 장치의 동작 원리를 배우고 반도체 기반 시스템 제어 능력 습득 |
| **AIoT 프로그래밍 실습** | Python, C/C++로 데이터 처리 및 AI 모델 적용 | 간단한 AI 모델을 IoT 디바이스에 적용하는 방법 익히기 |
| **반도체 회로 설계 기초** | 논리회로, Verilog/VHDL 기초, 시뮬레이션 툴 활용 | 반도체 칩 설계를 위한 디지털 회로 기초를 이해함 |
| **FPGA와 AI 응용** | FPGA 보드 실습, 경량 AI 모델 구현 | 하드웨어와 AI 알고리즘의 융합 경험 제공 |
| **AIoT 융합 프로젝트** | 자율주행카, 스마트팜, 헬스케어 시스템 설계 | 팀 기반 프로젝트로 실전형 문제 해결 능력 배양 |
| **반도체 산업과 진로** | 반도체 공정, 산업 트렌드, 윤리 교육 | 반도체 분야 진로 탐색 및 기술 윤리의식 함양 |

---
