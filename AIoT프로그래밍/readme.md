# AIoT 반도체 설계 과정 AIoT프로그래밍 과목 내용

AIoT 반도체 설계 과정에서 **AIoT프로그래밍** 과목은 IoT 장치 제어 기술과 경량 인공지능(AI) 알고리즘 적용 능력을 함께 배울 수 있는 실습 중심의 과목입니다. 이 과목은 센서 제어, 데이터 처리, 네트워크 통신 및 AI 모델을 IoT 디바이스에 적용하는 내용을 다룹니다.

---

## 📘 AIoT프로그래밍 과목 개요

| 구분 | 내용 |
|------|------|
| **과목명** | AIoT프로그래밍 |
| **목표** | IoT 장치에서 수집된 데이터를 프로그래밍적으로 처리하고, AI 알고리즘을 연계하여 지능형 서비스를 구현할 수 있도록 함 |
| **수준** | 고등학교 또는 전문대학/직업계열 고등학생 수준 |
| **운영 방식** | 실습 위주(60~70%), 프로젝트 기반 학습 포함 |

---

## 📚 세부 교육 내용

| 단원 | 주요 내용 | 실습 예 |
|------|-----------|----------|
| **1. IoT 개념과 시스템 구성** | IoT의 구조, 센서/액추에이터 개요, 통신 프로토콜 | Arduino 기반 온습도 센서 읽기 |
| **2. 프로그래밍 기초** | C/C++ 또는 Python 문법, 시리얼 통신 | LED 제어, 센서값 출력 |
| **3. 센서 제어 및 데이터 처리** | 아날로그/디지털 센서 제어, 데이터 필터링 | 초음파 센서, 조도 센서 제어 및 시각화 |
| **4. 통신 프로그래밍** | Wi-Fi, Bluetooth, MQTT 통신 | ESP32를 이용한 클라우드 데이터 전송 |
| **5. AI 기초 및 활용** | 머신러닝/딥러닝 기초, TinyML 소개 | 간단한 분류 모델을 IoT 장치에 탑재 |
| **6. AIoT 응용 프로그래밍** | AI 모델 추론, Edge 컴퓨팅 | 얼굴 인식, 동작 감지 등 AI 기능 탑재 |
| **7. 프로젝트 실습** | 팀별 AIoT 융합 프로젝트 | 스마트홈, 헬스케어, 농업 자동화 시스템 개발 |

- **Python (Micropython, Edge Impulse)**  
- **ESP32, Raspberry Pi, Jetson Nano**
- **Cloud 플랫폼**: Thingspeak, Firebase, AWS IoT 등
